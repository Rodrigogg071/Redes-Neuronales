# Redes-Neuronales

**Nombre:** Juan Rodrigo Guerrero De la Garza  
**Matrícula:** 611471  

**Objetivo:** Ser capaz de generar un modelo de aprendizaje profundo con redes neuronales.

---
En este proyecto se utiliza el dataset MNIST, un conjunto ampliamente empleado en visión computacional compuesto por 70,000 imágenes en escala de grises de 28×28 píxeles que representan dígitos escritos a mano del 0 al 9. De estas imágenes, 60,000 se utilizan para entrenamiento y 10,000 para evaluación, cada una acompañada de su etiqueta correspondiente. El dataset se obtiene automáticamente a través de Keras y proviene del repositorio original de Yann LeCun, disponible públicamente en Yann LeCun’s MNIST Database, donde fue presentado por primera vez en la publicación científica “Gradient-Based Learning Applied to Document Recognition”.
